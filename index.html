HTML Tags and Attributes

Table Tags

Tag Name Description
<table> </table> Table The container/wrapper for all of the data on a table.
<thead> </thead> Table header The tag used to define the column headers in a table.
<th> </th> Th The tag used to format data in the column header in a table.
<tfoot> </tfoot> Table Footer The tag used to define the bottom row of a table.
<tbody> </tbody> Table body The tag used to define the main body in a table.
<tr> </tr> Table Row The tag used to add in a row on a Table.
<td> </td> Table Data The tag used to add data into each table row (nested within each "tr").

Table Attributes

Attribute Name Description
colspan Column Span Makes table data span multiple columns on a table.
rowspan Row Span Makes table data span multiple cells in a table row.

Semantic HTML

Tag Name Description
<header> </header> Header Used for navigational links or introductory content.
<nav> </nav> Nav Used to define blocks of navigational links.
<main> </main> Main Used to define where the dominant type of content is on a webpage.
<footer> </footer> Footer Denotes the bottom of a webpage; usually contains contact info, copyright info, etc.
<article> </article> Article Defines content that can stand on its own. Can include: articles, blogs, etc.
<section> </section> Section Defines the start of a section, chapter, or groups parts of a webpage with similar themes.
<aside> </aside> Aside Used for contant this is relevant, but not necessary to the primary content of a webpage.
<figure> </figure> Figure Used to encapsulate image tags in your HTML code.
<figcaption> </figcaption> Figure Caption Used to put a caption on images on your website. It's special because it ensures that the text sticks to the image rather than change when the format changes.
<audio> </audio> Audio Used to embed audio files into your webpage.
<video> </video> Video Used to embed videos into your webpage.
<embed> Embed* *This is legacy code. Do not use.* A self-closing tag that can embed any external content onto your webpage.

General HTML Tags

Tag Name Description
<!--DOCTYPE html--> Document Type: HTML Used to tell your web browser what kind of code to expect. In this case, HTML.
<html> </html> HTML Used to denote where your HTML code will be
<body> </body> Body Used to declare where your main HTML code will be in.
<h1> </h1> Header 1 Used to display text in a header type font. Goes from h1 -> h6. H1 is largest, h6 is the smallest.
<p> </p> Paragraph HTML tag that lets you put text onto your webpage.
<div> </div> Division Used to create sections of similar content on a webpage. The non-semantic HTML version of "section".
<span> </span> Span Used to separate a section of text so that you can style particular parts of it.
<em> </em> Emphasize Used to Italicize text.
<strong> </strong> Bold Used to bold text.
<ul> </ul> Unordered List Used to start & contain all of the text in a bullet-point style list.
<ol> </ol> Ordered List Used to start & contain all of the text in a numbered list.
<li> </li> Add to List Used inside of <ul> or <ol> to add text or data to the respective list type.
<a> </a> Anchor Tag Used to create a link on your webpage/website. In between <a> & </a>, you can put in text, or an image.

General HTML Attributes

Attribute Name Description
class Class Class gives lines of code an identity that does not have to be unique.
id ID Used to give a singular line or section of code a unique identity.
src Source Used to attach a URL to an anchor tag, image, video, or audio file.
href Href used to link internally to your webpage.
alt Alternative Text Used to display alternative text in case an image, video, or audio file cannot play.
target Target Used to determine whether a link sends you to another tab or not.

CSS Cheat Sheet

CSS Selectors

Selector & Name Targeting Code Example
Asterisk: "*" Targets everything * {
text-align: center;
}
ID: "#" targets everything with a specified HTML ID #identity {
text-decoration: underline;
}
Class: "." targets everything with a specified HTML class .class {
line-spacing: 2em;
}
HTML Attributes: "[]" targets specific HTML attributes, or tags with a specific attribute. a[href] {
color: purple;
}
HTML tags: "[html tag]" targets HTML tags. h1 {
color: purple;
}
Stacking Selectors: "," Allows you to target multiple tags with the same CSS code. h1, p, li {
color: purple;
}

Display & Positioning

Property/Declaration Description Code Example
position
(static, relative, absolute, sticky) 		CSS code that changes the default block of positioning space set by HTML. There's static (default), relative, absolute, and sticky. p {
position: static;
}
relative position
(top, bottom, left, right) 		Allows you to move an HTML element relative to its default position (top, bottom, left, and right). p {
position: relative;
top: 40px;
left: 70px;
}
absolute position All other elements will ignore the specified element position, even in its parent space. Doing this will remove the element from the normal document flow. p {
position: absolute;
width:50%
}
fixed position Used to fix an element to a specific part of the page, often used for navigational bars. Doing this removes the element from the normal document flow. p {
position: fixed;
top:50%
}
sticky position Moves an element within its fixed position/container while not removing it from the normal document flow. p {
position: sticky;
right:50%
}
z-index CSS property that controls how far "forwards" or "backwards" an element will be displayed; like layers in a painting. Uses an integer value, the higher the value, the more "forward" and element will be. h2 {
z-index: 3;
}
display
(inline, block, inline-block) 		Every HTML element has a default display value that dictates if it can share horizontal space with other elements. To alter this, you use inline, block, or inline-block display.
inline display Adds a box that wraps tightly around the element, only giving it precisely the space it needs so that other elements can share horizontal space with it. NOTE: cannot be edited in CSS for size, height, and width. h1 {
display: inline;
}
block display Elements that take up the entire width of the page by default, but their width can be set in CSS. i.e., they take up their own line. h2 {
display: block;
}
inline-block display Combines inline & block displays, and the specified elements can appear beside each other, and adjusted in CSS. h2 {
display: inline-block;
}
float a CSS property that's used for wrapping text around an image. An element can be floated left, or right. But, the element must have its width specified first. h2 {
float: right;
width: 50%;
}
clear CSS property that specifies how elements should interact when they bump into eachother. There's left (left sides won't touch), right, both (neither side will touch), and none (all sides can touch). h2 {
clear: none;
}

The Box Model

Property/Declaration Description Code Example
height & width CSS property that alters the amount of height & width space an HTML element takes up. p {
height: 20%;
width: 550px;
}
min & max height and width sets either a minimum, or maximum, height and width that an element can take up. p {
min-height: 40%;
max-width: 550px;
}
border A line that surrounds and element that can be altered or shown with CSS. Uses the Width (W), Style (S), and Color (C) format. p {
border: W (2px), S (solid), C (red);
}
border-radius Used to round the edges on borders. Can be used after defining the border, or on its own. p {
border: W (2px), S (solid), C (red);
border-radius: 5px;
}
padding The space in between the content/elements and its border. uses a Top (T), Right (R), Bottom (B), and Left (L) model, or can be abbreviated with one number to represent all sides. p {
padding: 10px;
}
margin The space directly outside the border of an element. Decides how far away an element should be from one another. uses a TRBL model, or can be abbreviated. p {
margin: 10px;
}
auto Margin property that allows you to center content automatically p {
margin: auto;
}
overflow due to sizings and margins, elements can spill into each other (i.e., overflow). This CSS property tells the browser what to do when that happens. values: hidden, scroll, or visible. p {
overflow: hidden;
}
resetting defaults Browsers often have default margins and padding, so you target everything (*), and set them to 0, allowing you to manipulate elements as you please. * {
margin: 0;
padding: 0;
}
visibility Elements can be hidden from view with this property.Values: hidden, visible, collapse (collapses an element). p {
visibility: hidden;
}
changing the box model The box model is effective, but sometimes you need a little more control over how much space an element takes up. Normally, everything (padding, width, border, and margin) gets added to the total element width and height, but that may cause overflow. Therefore, you can use the box-sizing CSS property to alter that.
content-box This is the default Box Model that is set in browsers. * {
box-sizing: content-box;
}
border-box A box model where the width and height property is the actual rendered width of an element, including its borders and any padding. Benefit: it ensures that all widths and dimensions stay consistent. * {
box-sizing: border-box;
}

Improved CSS

Property/Declaration Description Code Example
Typography Typography is the ancient art of arranging text on a page. Digital Typography (what we're doing right now). is a relatively new practice.
fonts w/ multiple words When using a font with multiple words in it (e.g., Times New Roman), put the font name in single quotes (' '). * {
font-family: 'Times New Roman';
}
web-safe-fonts Are a list of fonts that are already available through most web browsers and computers. Ideal for Fallback Fonts.
fallback fonts & Font Stacks Fallback Fonts are alternative fonts that will be displayed should a computer not have your desired font. You add Fallback Fonts by creating a Font Stack (see example). Browsers will read left to right when determining what fonts to use. * {
font-family: 'Times New Roman', Arial, Helvetica, serif;
}
alternative font-weights When using the font-weight CSS property, you could use a keyword value (bold, normal, light), or, a numerical value from 1-1000. 400 is normal, 700 is bold. p {
font-weight: 700;
}
em A unit of measurement which indicates size relative to the current font size. 1em = same as current font size, 2em = twice the current font size, etc. p {
line-height: 1.5em;
}
line height. Adjusts the height of the line of text, measured in em. p {
line-height: 1.5em;
}
word spacing CSS property that adds space in between words. p {
word-spacing: 10px;
}
web fonts/custom fonts Web fonts are custom fonts that you download, and may not be on most computers. Used to add a special twist to your websites. A free resource is Google Fonts.
web fonts with <link> To add in a custom font, or web font, you can use <link> in the <head> section of your HTML document, then use the font-family CSS property as you would normally. HTML:
<head>
<link href='font url' rel='stylesheet'
</head>

CSS:
p {
font-family: 'imported font name';
}
web fonts with @font-face You can also import custom fonts without mixing your HTML & CSS code by using the @font-face CSS property. Start by downloading the font as above, then follow the coding example. NOTE: link your fonts in the following format order: WOFF2 -> WOFF -> TTP. After @font-face has been set up, simply use font-family with your custom font name to successfully import your custom font! @font-face {
font-family: 'insert custom font name';
src: url('font.woff2') format('woff2');
src: url('font.woff') format('woff)');
src url('font.ttp') format(truetype);
}
